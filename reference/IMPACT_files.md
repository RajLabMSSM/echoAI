# IMPACT files

URLs to annotations/LD-scores generated by
[IMPACT](https://github.com/immunogenomics/IMPACT) (Inference and
Modeling of Phenotype-related ACtive Transcription).

## Usage

``` r
data("IMPACT_files")
```

## Format

An object of class `data.table` (inherits from `data.frame`) with 88
rows and 9 columns.

## Source

` IMPACT_files <- downloadR::zenodo_list(token = token)[!endsWith(filename, c(".zip",".tbi")),] usethis::use_data(IMPACT_files, overwrite = TRUE) `
